# ✨ React 할일 관리 웹앱

직관적이고 모던한 디자인의 할일 관리 애플리케이션입니다.

## 🚀 빠른 시작

```bash
# 브라우저에서 바로 열기
start todo-app.html
```

또는 `todo-app.html` 파일을 더블클릭하세요!

## 📸 주요 화면

### 대시보드
- 📊 오늘의 할일 통계를 한눈에 확인
- 완료/미완료/전체 개수 실시간 업데이트

### 할일 카드
- 💼 업무 (파스텔 블루)
- 🏠 개인 (파스텔 핑크)  
- 📚 공부 (파스텔 그린)

## ⚡ 핵심 기능

| 기능 | 설명 |
|------|------|
| 📝 추가/수정/삭제 | 완벽한 CRUD 기능 |
| ✅ 완료 체크 | 체크박스로 간편하게 |
| 📅 날짜/시간 | 마감일 설정 가능 |
| 📂 폴더 관리 | 3개 폴더로 분류 |
| 🔍 필터링 | 탭과 폴더로 검색 |
| 💾 자동 저장 | LocalStorage 사용 |
| 📱 반응형 | 모든 기기 지원 |

## 🎨 디자인 특징

- **흰색 배경** + **파스텔 톤** 포인트 컬러
- **부드러운 애니메이션** (hover 효과, 페이드인)
- **폴더별 색상 구분**으로 직관적인 관리
- **모던한 그라디언트** 버튼과 헤더

## 📱 반응형 레이아웃

```
Desktop (1024px+)   → 3컬럼 그리드
Tablet (768-1023px) → 2컬럼 그리드  
Mobile (~767px)     → 1컬럼 스택
```

## 🛠️ 기술 스택

```
React 18 (CDN)
+ Babel Standalone
+ LocalStorage API
+ Pure CSS
= 단일 HTML 파일 (1,013 lines)
```

## 💡 사용 팁

1. **제목만 입력**해도 바로 추가 가능
2. **날짜 설정**하면 자동으로 탭에 분류
3. **폴더 색상**으로 한눈에 구분
4. **메모**에 상세 내용 추가
5. **체크박스** 클릭으로 빠른 완료
