# 쉐어하우스 규칙 관리 앱 - 빠른 시작 가이드

## 🚀 앱 실행하기

### Windows에서 실행:
```bash
cd c:\py
python app.py
```

### 브라우저에서 접속:
앱이 시작되면 브라우저를 열고 다음 주소로 접속하세요:
- http://localhost:5000
- 또는 http://127.0.0.1:5000

## ✨ 주요 기능 테스트

### 1. 규칙 추가 테스트
- 좌측 폼에 다음을 입력해보세요:
  - 제목: "밤 10시 이후 조용히 하기"
  - 설명: "밤 10시부터 아침 8시까지는 큰 소리를 내지 않습니다"
  - 카테고리: 소음
  - 중요도: 높음
- "규칙 추가" 버튼 클릭
- 우측에 새 규칙 카드가 나타나는지 확인

### 2. 카테고리 필터 테스트
- 상단의 카테고리 버튼들을 클릭해보세요
- 선택한 카테고리의 규칙만 표시되는지 확인

### 3. 규칙 수정 테스트
- 규칙 카드 하단의 "수정" 버튼 클릭
- 모달 창이 열리면 내용을 수정
- "저장" 버튼 클릭하여 변경사항 저장

### 4. 규칙 삭제 테스트
- 규칙 카드 하단의 "삭제" 버튼 클릭
- 확인 메시지에서 "확인" 클릭
- 규칙이 목록에서 사라지는지 확인

## 🔧 문제 해결

### 앱이 시작되지 않는 경우:
1. Python이 설치되어 있는지 확인:
   ```bash
   python --version
   ```

2. 필요한 패키지가 설치되어 있는지 확인:
   ```bash
   pip install -r requirements.txt
   ```

### 포트 5000이 이미 사용 중인 경우:
1. `app.py` 파일을 열어서 마지막 줄을 수정:
   ```python
   app.run(debug=True, host='0.0.0.0', port=5001)
   ```
2. 브라우저에서 http://localhost:5001 로 접속

### 데이터베이스 오류가 발생하는 경우:
1. `sharehouse.db` 파일을 삭제
2. 앱을 다시 시작하면 자동으로 새 데이터베이스가 생성됩니다

## 📱 반응형 디자인

- 중요도에 따라 규칙 카드의 왼쪽 테두리 색상이 다르게 표시됩니다
  - 빨강: 높음
  - 주황: 보통
  - 초록: 낮음
- 규칙을 추가하면 상단에 알림이 표시됩니다
- 모든 데이터는 자동으로 저장되어 앱을 다시 시작해도 유지됩니다
