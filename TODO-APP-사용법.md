# 📝 할일 관리 (To-Do Manager) 사용 가이드

React 기반의 모던하고 직관적인 할일 관리 웹앱입니다.

## 🚀 실행 방법

1. **간단한 실행**: `todo-app.html` 파일을 브라우저에서 더블클릭하여 엽니다.
2. **또는**: 브라우저에서 `todo-app.html` 파일을 드래그 앤 드롭합니다.

별도의 설치나 서버 실행이 필요 없습니다! 😊

## ✨ 주요 기능

### 1️⃣ 오늘의 일정 대시보드
- 상단에서 오늘의 할일, 완료된 할일, 남은 할일, 전체 할일 개수를 한눈에 확인
- 실시간으로 업데이트되는 통계

### 2️⃣ 할일 추가
- **제목**: 할일의 이름 (필수)
- **메모**: 상세한 설명이나 부가 정보
- **날짜**: 마감일 설정
- **시간**: 구체적인 시간 설정
- **폴더**: 업무 💼 / 개인 🏠 / 공부 📚 중 선택

### 3️⃣ 카테고리 탭
- **📅 오늘**: 오늘 날짜의 할일만 표시
- **📆 이번주**: 이번 주(월~일)의 할일 표시
- **🗓️ 나중에**: 이번 주 이후의 할일 표시
- **📋 전체**: 모든 할일 표시

### 4️⃣ 폴더 관리
- **전체**: 모든 폴더의 할일 표시
- **💼 업무**: 업무 관련 할일 (파스텔 블루)
- **🏠 개인**: 개인적인 할일 (파스텔 핑크)
- **📚 공부**: 학습 관련 할일 (파스텔 그린)

### 5️⃣ 할일 관리
- **완료 체크**: 체크박스를 클릭하여 완료/미완료 전환
- **수정**: 할일 카드의 "수정" 버튼 클릭
- **삭제**: "삭제" 버튼 클릭 (확인 메시지 표시)

## 🎨 디자인 특징

### 색상 시스템
- **흰색 배경**: 깔끔하고 모던한 느낌
- **파스텔 블루** (#B8D4FF): 업무 폴더
- **파스텔 핑크** (#FFB8D4): 개인 폴더
- **파스텔 그린** (#B8FFD4): 공부 폴더
- **그라디언트**: 버튼과 헤더에 세련된 그라디언트 효과

### 애니메이션 효과
- 카드 hover 시 부드러운 상승 효과
- 그림자 변화 애니메이션
- 페이드인 애니메이션으로 자연스러운 등장

## 📱 반응형 디자인

### 데스크톱 (1024px 이상)
- 3컬럼 그리드 레이아웃
- 넓은 화면 활용

### 태블릿 (768px ~ 1023px)
- 2컬럼 그리드 레이아웃
- 최적화된 간격

### 모바일 (767px 이하)
- 1컬럼 스택 레이아웃
- 터치 친화적인 버튼 크기
- 세로로 나열된 필터 버튼

## 💾 데이터 저장

- **LocalStorage 사용**: 브라우저에 자동으로 데이터 저장
- **자동 저장**: 할일 추가/수정/삭제 시 즉시 저장
- **새로고침 후에도 유지**: 브라우저를 닫았다가 열어도 데이터 보존

## 🔧 사용 팁

1. **빠른 추가**: 제목만 입력하고 추가 버튼을 눌러도 OK!
2. **폴더 활용**: 할일을 업무/개인/공부로 구분하면 관리가 편해요
3. **날짜 설정**: 마감일을 설정하면 탭으로 쉽게 필터링 가능
4. **시간 설정**: 중요한 약속은 시간까지 설정하세요
5. **메모 활용**: 상세한 내용은 메모에 적어두세요

## 🛠️ 기술 스택

- **React 18**: 최신 React 버전 (CDN)
- **Babel Standalone**: JSX를 브라우저에서 실행
- **LocalStorage API**: 데이터 영구 저장
- **순수 CSS**: 외부 라이브러리 없이 구현
- **Responsive Design**: 모든 디바이스 지원

## 📂 프로젝트 구조

```
todo-app.html (단일 파일)
├── HTML 구조
├── CSS 스타일
└── React 컴포넌트
    ├── App (메인)
    ├── Dashboard (대시보드)
    ├── TabBar (탭)
    ├── FolderBar (폴더)
    ├── TodoForm (추가/수정 폼)
    ├── TodoList (목록)
    └── TodoItem (개별 항목)
```

## 🎯 사용 시나리오

### 아침 루틴
1. "오늘" 탭을 클릭하여 오늘의 할일 확인
2. 대시보드에서 완료해야 할 할일 개수 파악
3. 우선순위가 높은 할일부터 체크

### 새 할일 추가
1. 상단 폼에서 제목 입력
2. 필요시 메모, 날짜, 시간 입력
3. 적절한 폴더 선택
4. "추가하기" 버튼 클릭

### 주간 계획
1. "이번주" 탭을 클릭
2. 이번 주에 해야 할 일들을 한눈에 확인
3. 폴더별로 필터링하여 업무/개인/공부 분리

## 🌟 장점

✅ 설치 불필요 - 브라우저만 있으면 OK  
✅ 오프라인 작동 - 인터넷 연결 불필요  
✅ 자동 저장 - 데이터 손실 걱정 없음  
✅ 직관적인 UI - 누구나 쉽게 사용  
✅ 반응형 - 모든 기기에서 사용 가능  
✅ 빠른 속도 - 즉각적인 반응  
✅ 깔끔한 디자인 - 눈이 편안함  

## 🎨 색상 커스터마이징

HTML 파일의 CSS 섹션에서 색상을 쉽게 변경할 수 있습니다:

```css
/* 업무 폴더 색상 */
.folder-button.work { border-color: #B8D4FF; }

/* 개인 폴더 색상 */
.folder-button.personal { border-color: #FFB8D4; }

/* 공부 폴더 색상 */
.folder-button.study { border-color: #B8FFD4; }
```

## 💡 참고사항

- Chrome, Firefox, Safari, Edge 등 모던 브라우저에서 작동
- 브라우저의 LocalStorage를 사용하므로 데이터는 해당 브라우저에만 저장됨
- 다른 기기에서 접근하려면 파일을 복사하여 열면 됨
- LocalStorage를 지우면 데이터가 삭제되니 주의!
